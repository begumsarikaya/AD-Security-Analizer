# AD Security Analyzer

AD Security Analyzer is a desktop application developed in Python for analyzing security vulnerabilities in Active Directory environments.  
The application connects to Windows Server Active Directory services using LDAP and performs user, password, and account-based security analysis.

## Features

- Active Directory user analysis
- Detection of disabled accounts
- Detection of password never expires accounts
- Identification of inactive users
- LDAP connection support
- User and group information listing
- Security-focused account reporting
- Simple and user-friendly interface

## Technologies Used

- Python
- LDAP3
- Tkinter
- Windows Server 2019
- Active Directory Domain Services (AD DS)

## Project Purpose

The purpose of this project is to improve the security management process in Active Directory environments by identifying weak or risky user accounts and helping administrators analyze domain security more efficiently.
